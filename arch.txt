новая игра 
    каждый раз берем из кук айди игрока и шлем хелло на сервер
        на сервере: 
            если айди нулл 
                на сервере игрока искать не будут, шлем все геймсэты с флагом isStart === false
            если есть 
                выбираем игры без игрока этого. Обновляем сокет айди в таблице онайл. 
            и шлем кленту ответку хелло
    если в ответе ноунейм 
        показываем форму авторизации / регистрации, список игроков
    иначе
        показываем форму конструктор игры, список игроков


    авторизация
        клиент
            шлем имя игрока, и пароль из формы
        сервер
            проверяем в базе - получаем игрока. Записываем в таблицу онлайн игроки. Смотрим у игрока currentMeetingId. Если есть то находим эту встречу. 
            Шлем юзеру игрока и эту встречу
        клиент
            если пришла встреча и у нее isStart === 1
                предлагаем продолжить. Если продолжить то шлем на сервер запрос продолжить !!!1!!!
            если пришла встреча и у нее isStart === 0
                показываем индикатор ожидания, и предлагаем удалить встречу !!!2!!!
        сервер на запрос продолжить игру
            смотрит есть ли в онлайне второй игрок, если есть - шлем запрос второму игроку, если второй игрок отказался то шлем отказ, 
            и автоматом первый игрок выиграл
        
    создание игры
        клиент
            выбираем тип игры и шлем на сервер
        сервер 
            записываем в базу
                игру - тип игры, в players пушим игрока { айди игрока первого, ранг 1 }, whoseTurn = 0
                тип встречи, пушим в score { айди первого игрока, каунт 0 }, пушим в games айди игры, isStart = 0
                шлем клиенту что игра создана !!!3!!!
                шлем клиентам что новая игра пришла !!!4!!!
